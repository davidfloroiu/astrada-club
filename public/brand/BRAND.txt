ASTRADA CLUB — Logo Package
============================

VECTOR (SVG) — scalable masters, use for print, web, signage
  astrada-primary-navy.svg        Primary lockup, platinum mark on deep navy
  astrada-primary-light.svg       Primary lockup, navy on warm paper
  astrada-primary-transparent.svg Primary lockup, navy ink on transparent bg
  astrada-horizontal.svg          Horizontal lockup (mark + wordmark side by side)
  astrada-mark-navy.svg           Monogram only, navy
  astrada-mark-reversed.svg       Monogram only, light (for dark backgrounds)
  astrada-mark-platinum.svg       Monogram only, platinum gradient

RASTER (PNG) — quick-use exports
  png/astrada-primary-navy.png
  png/astrada-primary-light.png
  png/astrada-horizontal.png

COLORS
  Deep Navy    #0E1B30
  Platinum     #C5CAD3  (gradient: #F3F5F8 -> #AEB5C0 -> #9AA1AD)
  Warm Paper   #EDEBE5

TYPE
  Wordmark / display   Cormorant Garamond (Medium / 500)
  Supporting / caption Hanken Grotesk

NOTES
  - The SVG wordmark references the Cormorant Garamond web font. For final
    print/handoff artwork, convert the wordmark text to outlines in a vector
    editor (Illustrator/Figma: "Create outlines" / "Outline stroke") so the
    files are fully self-contained.
  - The monogram (mark) SVGs are pure vector paths — fully portable, no fonts.
